# IBM Data Science Professional Certificate

Este repositorio contiene los módulos, ejercicios y el proyecto final realizados para obtener la certificación profesional de **IBM Data Science**.

## 🎓 Sobre la Certificación
El programa cubre las herramientas y metodologías más utilizadas en la industria:
*   **Lenguajes**: Python y SQL.
*   **Librerías**: Pandas, NumPy, Matplotlib, Seaborn, Folium y Scikit-Learn.
*   **Metodología**: De la comprensión del negocio a la implementación del modelo.

## 🚀 Proyecto Final (Applied Data Science Capstone)
El trabajo final (`Final-Project-Data-Science-IBM`) integra todas las habilidades adquiridas:
*   **Caso de Estudio**: Predicción del éxito en el aterrizaje de las primeras etapas del cohete Falcon 9 de **SpaceX**.
*   **Técnicas utilizadas**: 
    1.  Recolección de datos (Web Scraping y APIs).
    2.  Análisis con SQL para filtrar insights.
    3.  Visualización interactiva con mapas (Folium) y Dashboards (Plotly Dash).
    4.  Desarrollo de modelos de clasificación para predecir si el aterrizaje será exitoso.
